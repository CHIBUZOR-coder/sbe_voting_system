# Public
GET    /api/orgs                     ← list all approved orgs (paginated + search)
GET    /api/orgs/:slug               ← single org by slug

# SUPER_ADMIN only
GET    /api/orgs/admin/pending       ← all orgs waiting for approval
PATCH  /api/orgs/:id/approve         ← approve an org
PATCH  /api/orgs/:id/reject          ← reject an org

# Verified users
POST   /api/orgs                     ← create org (logo optional)
PATCH  /api/orgs/:id                 ← update org name/description/logo

# Member management
GET    /api/orgs/:id/members         ← list members (members only)
POST   /api/orgs/:id/members         ← add member by email
DELETE /api/orgs/:id/members/:userId ← remove member
